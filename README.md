# Prey API v2

Сервис на **Python / Flask**, предоставляющий REST API с данными по вселенной игры *Prey* (2017). Проект агрегирует информацию о тифонах (hostiles), способностях (abilities), чипсетах (chipsets), локациях космической станции «Талос-1» (locations) и доступном вооружении (weapons).

Данный сервис разработан в качестве бэкенда для интеграции с фронтенд-приложениями.

---

## Стек технологий

* **Язык программирования:** Python 3.10
* **Веб-фреймворк:** Flask
* **База данных:** SQLite 3 (легковесная реляционная БД, хранящаяся в файле `prey.db`)

---

## Архитектурные особенности кода

1.  **Потокобезопасность SQLite в Flask:** Подключение к базе данных настроено с флагом `check_same_thread=False`. Это позволяет Flask безопасно обрабатывать запросы в многопоточном режиме, используя единый глобальный курсор.
2.  **Защита от SQL-инъекций:** Все эндпоинты, принимающие параметры (например, выборка по `id`), используют параметризованные SQL-запросы (`?`), что полностью исключает возможность внедрения вредоносного SQL-кода.
3.  **Динамический маппинг в JSON:** Для эндпоинтов получения конкретной сущности используется автоматическая сборка словаря через `cursor.description`. Это позволяет коду оставаться гибким к изменению количества и порядка колонок в таблицах БД.
4.  **Безопасность (CORS):** Подключено расширение `flask_cors`, снимающее ограничения браузеров на запросы с других хостов.
